## To Run on local pc
**Step 1:**
Run this command in VS code after opening the desired folder.
```bash
git clone https://github.com/aafthxb/Parallax.git
```
**Step 2:**
Then run.
```bash
cd Parallax
```
**Step 3:**
Then run.
```bash
npm install
```
**Step 4:**
Then run.
```bash
npm run dev
```
**Step 5**
Then open the local host link.

Woalla you just ran this website locally on your pc.
